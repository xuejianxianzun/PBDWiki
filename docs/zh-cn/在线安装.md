# 在线安装

## PC

推荐使用的浏览器：
- Chrome
- Edge
- Firefox

Chrome、Edge 等 Chromium 内核的浏览器可以从 [Chrome Web Store](https://chrome.google.com/webstore/detail/powerful-pixiv-downloader/dkndmhgdcmjdmkdonmbgjpijejdcilfh) 安装本扩展。

Firefox 浏览器可以从 [Add-Ons](https://addons.mozilla.org/firefox/addon/powerfulpixivdownloader/) 安装本扩展。

?>如果你喜欢这个扩展，请给它评分，使它可以被更多人看到，谢谢！

!>注意：
- 请使用各浏览器的最新版本。
- 不建议使用其他浏览器。虽然有些浏览器也是基于 chromium 内核的，但是它们的内核版本可能比较旧，或者对内核进行了一些魔改，可能会影响下载器的正常使用。

## Android


现在可以使用 Microsoft Edge Canary 浏览器离线安装这个扩展，具体步骤请查看：

[在 Microsoft Edge Canary 浏览器上安装](/zh-cn/MicrosoftEdgeCanary.md)

*以前 Android 上的 Kiwi 浏览器可以在线安装这个扩展，但现在它已经停止开发并且从 Google Play 下架，因此不再推荐。*

------------

Android 上的 Firefox 浏览器虽然可以安装扩展，但似乎只能安装有“为移动端优化”标记的扩展。目前这个扩展程序不在此列表里，所以无法安装和使用。

## IOS

!>无法使用

IOS 上的 Safari 浏览器似乎只能从 App Store 里安装扩展，所以它无法使用这个扩展。
